Installationsanweisungen für die Microsoft Visual Studio LightSwitch-Anwendung

 Die folgenden Schritte sind notwendig, um eine Desktopclientanwendung mit zwei Ebenen zu installieren:

1.    Installieren Sie SQL Server Express auf dem Zielcomputer (Client), wenn nicht bereits geschehen. LightSwitch-Anwendungen erfordern SQL Server 2005 oder höher bzw. SQL Server Express 2005 oder höher. Weitere Informationen über die Installationsoptionen von SQL Server Express

2.    Stellen Sie auf dem Zielcomputer mit dem Befehlstool sqlcmd.exe das Datenbankschema bereit. Das Tool Sqlcmd.exe wird mit SQL Server installiert, normalerweise im Verzeichnis "\Microsoft SQL Server\110\Tools\Binn" unter dem Verzeichnis "Programme" bzw. "Programme (x86)".

a.     Führen Sie in einer Eingabeaufforderung mit erweiterten Berechtigungen eine Änderung im Verzeichnis mit der Veröffentlichungsausgabe durch, und geben Sie einen Befehl ähnlich dem folgenden ein:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Ersetzen Sie MyApplication durch den Namen Ihrer Anwendung, und ersetzen Sie .\SQLExpress durch den Datenbankspeicherort auf dem Server.

3.    Lokalisieren Sie die web.config-Datei im Unterverzeichnis "Anwendungsdateien" der Veröffentlichungsausgabe.

a.     Öffnen Sie "web.config" in Notepad, und bearbeiten Sie die Verbindung im Abschnitt <connectionStrings> für die Verwendung Ihres Servers und Datenbanknamens, z. B. für den Eintrag ähnlich dem folgenden:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Ersetzen Sie SERVERNAME durch den Namen des Datenbankservers, und ersetzen Sie MyApplication durch den Namen Ihrer Anwendung.

4.    Führen Sie "Setup.exe" auf dem Zielcomputer aus, um die Anwendung zu installieren.
